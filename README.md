# Projeto de consumo de api via cep com javascript.
### Tecnologias usadas:
- Html
- Css
- Bootstrap
- Javascript


### Como funciona:
 Ao preencher seu cep no formulario a api automaticamente extrai seus dados para os campos : endereço,bairro,cidade e estado.
